My GIS Website Project

Introduction to GIS

Geographic Information System (GIS) is a technology that allows users to collect, manage, and analyze spatial data. It is widely used in various fields such as urban planning, environmental studies, and transportation. This project aims to demonstrate how GIS data can be displayed on a simple website .

Project Description

The GIS website is designed to present geographic information in a structured format. The website includes a static map and a data table that lists different locations with their corresponding latitude and longitude values.
